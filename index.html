PlayAroundMe Calculator

GitHub

8/2020

PlayAroundMe Calculator is a tool used to better learn new Magic: the Gathering cards as you play in Limited format events. The application calculates all permutations of resources your opponent has available and returns a list of cards they could potentially have in-hand so that you may "play around" those cards.

Build with React and JS ES6; Redux implemented for state management. Scryfall API utilized to populate new sets.
